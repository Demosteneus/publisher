Lengths

The lengths can be given in one of these units: pp, pc, in, pt, cm, mm, dd, cc.

Unit Description
pp Pica Point
pc pica (12 pp)
in inch (72.27 pp)
pt Big point (72 bp = 1 in) PostScript point, DTP-point
cm Centimeter
mm Millimeter
dd Didot point
cc cicero (12 dd)
