Elementname: Absatz

Beschreibung

Erzeugt einen Absatz innerhalb eines Textblocks.

Erlaubte Attribute: erlaubeumbruch, farbe, schriftart, sprache, textformat

Kindelemente: A, Aktion, B, Bild, Br, Fallunterscheidung, FürAlle, HLeerraum, I, Schleife, Schriftart, Sub, Sup, U, URL, Wert

Elternelemente: Bis, Fall, FürAlle, Schleife, Solange, Sonst, Td, Text, Textblock, Zuweisung

Attribute

erlaubeumbruch (Text, optional)

(Experimentell) Liste der Zeichen, bei denen ein Umbruch möglich ist.

farbe (Text, optional)

Farbe des Absatzes. Muss vorher mit DefiniereFarbe definiert worden sein.

schriftart (Text, optional)

Name der Schriftart. Voreinstellung ist »text« (mit kleinem t).

sprache (optional)

Name der Sprache für die Silbentrennung.

Deutsch:

Deutsche Trennmuster

Englisch (Großbritannien):

Englische Trennmuster auswählen

Französisch:

Französische Trennmuster
textformat (Text, optional)

Name des zu benutzenden Textformats. Wird kein Textformat angegeben, nimmt das System das Textformat text

Beispiel

<Textblock breite="27">
  <Absatz schriftart="Überschrift">
    <Wert>Hallo Welt</Wert>
  </Absatz>
</Textblock>

Beispiel

<Textblock breite="27">
  <Absatz schriftart="Überschrift" sprache="Deutsch">
    <Wert auswahl="node()"/>
  </Absatz>
  </Textblock>

wobei in den Daten ein Text steht, beispielsweise falls der aktuelle Knoten das Element Datensatz ist: 

<datensatz>Produktbeschreibung: die Artikelnummer <i>1234</i> setzt sich aus ... </datensatz>
