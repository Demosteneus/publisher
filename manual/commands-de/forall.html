Elementname: FürAlle

Beschreibung

Ruft die Kindelemente für alle Daten-Elemente auf, die auf das Muster bei auswahl passen.

Erlaubte Attribute: auswahl
Kindelemente: B, Bild, Fallunterscheidung, I, Schriftart, Sub, Sup, U, URL, Wert, Zuweisung
Elternelemente: Tabelle

Attribute

auswahl (XPath-Ausdruck)

Wählt die Kindelemente aus.

Beispiel

<Datensatz element="data">
  <ObjektAusgeben>
    <Tabelle breite="10">
      <FürAlle auswahl="eintrag">
        <Tr><Td><Absatz><Wert auswahl="string(.)"></Wert></Absatz></Td></Tr>
      </FürAlle>
    </Tabelle>
  </ObjektAusgeben>
</Datensatz>

Erzeugt eine Tabellenzeile für alle Elemente eintrag im Datensatz data. So müsste in etwa ein entsprechender Eintrag in den Daten aussehen: 

<data>
  <eintrag>a</eintrag>
  <eintrag>b</eintrag>
  <eintrag>c</eintrag>
</data>
