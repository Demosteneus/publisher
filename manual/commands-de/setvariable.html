Elementname: Zuweisung

Beschreibung

Weist einer Variablen einen Wert zu. Der Wert kann aus mehreren Elementen bestehen (siehe zweites Beispiel).

Erlaubte Attribute: auswahl, variable

Kindelemente: Absatz, Attribut, Element, Fallunterscheidung, FürAlle, Kopie-von, Schleife, Sequenz, SortiereSequenz, Spalten, Tabellenkopf, Td, Tlinie, Tr, Wert

Elternelemente: BeiSeitenAusgabe, BeiSeitenErzeugung, Bis, Datensatz, Fall, FürAlle, Include, Inhalt, Layout, Schleife, SeitenSpeichern, Solange, Sonst

Attribute

auswahl (XPath-Ausdruck, optional)

Inhalt, der der Variablen zugewiesen wird.

variable (Text)

Name der Variablen, der etwas zugewiesen werden soll.

Bemerkungen

Variablen haben eine globale Sichtbarkeit.

Beispiel

<Datensatz element="produkt">
  <Zuweisung variable="Textbreite" auswahl="5"/>
  <ObjektAusgeben spalte="1">
    <Textblock breite="{ $Textbreite }">
      <Absatz>
        <Wert auswahl="$Artikelnummer"/>
      </Absatz>
    </Textblock>
  </ObjektAusgeben>
</Datensatz>

Folgendes Beispiel zeigt, wie mehrere Absätze in einer Variablen gespeichert werden, um sie anschließend in einem Textblock auszugeben. 

<Datensatz element="produkte">
  <Zuweisung variable="Artikeltext"/>
  <BearbeiteKnoten auswahl="artikel"/>
  <ObjektAusgeben spalte="1">
    <Textblock breite="10">
      <Wert auswahl=" $Artikeltext "/>
    </Textblock>
  </ObjektAusgeben>
</Datensatz>

<Datensatz element="artikel">
  <Zuweisung variable="Artikeltext">
    <!-- Der vorherige Inhalt wird hinzugefügt -->
    <Wert auswahl="$Artikeltext"/>
    <Absatz>
      <Wert auswahl=" @beschreibung " />
    </Absatz>
  </Zuweisung>
</Datensatz>
