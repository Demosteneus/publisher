speedata Publisher by pgundlach

The speedata Publisher is a full featured database publishing software. It has a built-in layout description language which allows you to create arbitrary complex layouts.

Requirements

The speedata Publisher needs a LuaTeX binary with the Lua libxml2 bindings from asbradbury.org. You can at your choice create a single binary or load the Lua bindings at runtime. There is a github repository with a patched version of LuaTeX. That is what I use. You can compile it if you have the libxml2 headers installed.

Install

The contents of this repository is mostly ready to run. Put the sdluatex binary (see section requirements) in the bin/ directory and run "sprun" in your terminal window.
